# nifi-atlas
A bridge to Apache Atlas for provenance metadata created from  data transformations completed by Apache NiFi .

## Getting Started

> instructions for setting up working demo on dev environment


### Prerequisites 

• Java 8
•	Apache Atlas 0.8
•	Apache NiFi 1.3+
